Note:

Recommended text editors/IDEs:

1) VSCode - recommended

2) Sublime Text/Atom

3) (not recommended) Replit

This is the basic format of all HTML documents:

             
              <!DOCTYPE HTML>
              <html>
              <head>
                <title>title</title>
              <head>
              <body>
                <!--other stuff here-->
              </body>
              </html>
Practice: make a simple HTML page with one line per requirement: 
             
              1) put the necessary parts of an HTML page
              biggest header
              smaller header
              paragraph
              line break
              insert any image with alt text and resize it to 20 by 20 pixels
              bold
              strong
              italic
              emphasized
              highlight
              superscript
              subscript
              strikethrough
              insertion
              small text
              text color
              comic sans ms font
              text size 15px
              align text to the right
              background color red
              green text
              3px blue border
              ordered list with 3 items, roman numerals
              unordered list with disk bullets
              description list:
              color 1 - blue
              color 2 - green
              color 3 - red
              nest the unordered list in the ordered list
Worksheet answer key 
            
              <!DOCTYPE html>
              <html>
              <head><title>Reviewer 2</title></head>
              <body>
              <h1>biggest header</h1>
              <h2>smaller header</h2>
              <p>paragraph</p>
              <br>
              <img src="imagelink.png" alt="Invalid image" length="20px" width="20px">
              <b>bold</b>
              <strong>strong</strong>
              <i>italic</i>
              <em>emphasized</em>
              <mark>highlight</mark>
              <sup>superscript</sup>
              <sub>subscript</sub>
              <del>strikethrough</del>
              <ins>insertion</ins>
              <small>small text</small>
              <p style="color: blue;">text color</p>
              <p style="font-family: comic sans ms;">comic sans ms font</p>
              <p style="font-size: 15px;">text size 15px</p>
              <p style="text-align: right;">align text to the right</p>
              <p style="background-color: red;">background color red</p>
              <p style="color: green;">green text</p>
              <p style="border: 3px solid blue;">3px blue border</p>
              <ol type="I">
                <li>ordered list with 3 items, roman numerals</li>
                <li>second</li>
                <li>third</li>
                <li>
                  <ul list-style-type="disc">
                    <li>unordered list with disk bullets</li>
                  </ul>
                </li>
              </ol>
              <dl>
                <dt>Color 1</dt>
                <dd>-blue</dd>
                <dt>Color 2</dt>
                <dd>-green</dd>
                <dt>Color 3</dt>
                <dd>-red</dd>
              </dl>
              </body></html>
